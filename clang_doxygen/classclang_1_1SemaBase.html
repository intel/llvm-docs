◆ DeviceDiagnosticReason

enum clang::SemaBase::DeviceDiagnosticReason
strong

Bitmask to contain the list of reasons a single diagnostic should be emitted, based on its language.

This permits multiple offload systems to coexist in the same translation unit.

Enumerator
None 

Diagnostic doesn't apply to anything.

Included for completeness, but should make this a no-op.

OmpDevice 

OpenMP specific diagnostic.

OmpHost 
OmpAll 
CudaDevice 

CUDA specific diagnostics.

CudaHost 
CudaAll 
Sycl 

SYCL specific diagnostic.

Esimd 

ESIMD specific diagnostic.

All 

A flag representing 'all'.

This can be used to avoid the check all-together and make this behave as it did before the DiagnosticReason was added (that is, unconditionally emit). Note: This needs to be updated if any flags above are added.
