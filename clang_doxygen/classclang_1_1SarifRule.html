Detailed Description

A SARIF rule (reportingDescriptor object) contains information that describes a reporting item generated by a tool.

A reporting item is either a result of analysis or notification of a condition encountered by the tool. Rules are arbitrary but are identifiable by a hierarchical rule-id.

This builder provides an interface to create SARIF reportingDescriptor objects via the SarifRule::create static method.
