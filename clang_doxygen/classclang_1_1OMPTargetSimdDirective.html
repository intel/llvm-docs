Detailed Description

This represents '#pragma omp target simd' directive.

#pragma omp target simd private(a) map(b) safelen(c)

In this example directive '#pragma omp target simd' has clauses 'private' with the variable 'a', 'map' with the variable 'b' and 'safelen' with the variable 'c'.
