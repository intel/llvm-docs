Detailed Description

template<typename BaseTy, typename RegionTy>
class cl::sycl::ext::intel::experimental::esimd::simd_view< BaseTy, RegionTy >

This class represents a reference to a sub-region of a base simd object.

The referenced sub-region of the base object can be read from and written to via an instance of this class.
