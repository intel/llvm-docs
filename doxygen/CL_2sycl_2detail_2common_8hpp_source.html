156 // Helper function for extracting implementation from SYCL's interface objects.
157 // Note! This function relies on the fact that all SYCL interface classes
158 // contain "impl" field that points to implementation object. "impl" field
159 // should be accessible from this function.
160 //
161 // Note that due to a bug in MSVC compilers (including MSVC2019 v19.20), it
162 // may not recognize the usage of this function in friend member declarations
163 // if the template parameter name there is not equal to the name used here,
164 // i.e. 'Obj'. For example, using 'Obj' here and 'T' in such declaration
165 // would trigger that error in MSVC:
