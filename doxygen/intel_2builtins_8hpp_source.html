24 // Provides functionality to print data from kernels in a C way:
25 // - On non-host devices this function is directly mapped to printf from
26 // OpenCL C
27 // - On host device, this function should be equivalent to standard printf
28 // function from C/C++.
29 //
30 // Please refer to corresponding section in OpenCL C specification to find
31 // information about format string and its differences from standard C rules.
32 //
33 // This function is placed under 'experimental' namespace on purpose, because it
34 // has too much caveats you need to be aware of before using it. Please find
35 // them below and read carefully before using it:
36 //
37 // - According to the OpenCL spec, the format string must be
38 // resolvable at compile time i.e. cannot be dynamically created by the
39 // executing program.
40 //
41 // - According to the OpenCL spec, the format string must reside in constant
42 // address space. This requires to perform "tricky" declarations of them, see
43 // test/built-ins/printf.cpp for examples
44 // FIXME: this potentially can be done on SYCL FE side automatically
45 //
46 // - The format string is interpreted according to the OpenCL C spec, where all
47 // data types has fixed size, opposed to C++ types which doesn't guarantee
48 // the exact width of particular data types (except, may be, char). This might
49 // lead to unexpected result, for example: %ld in OpenCL C means that printed
50 // argument has 'long' type which is 64-bit wide by the OpenCL C spec. However,
51 // by C++ spec long is just at least 32-bit wide, so, you need to ensure (by
52 // performing a cast, for example) that if you use %ld specifier, you pass
53 // 64-bit argument to the cl::sycl::experimental::printf
54 //
55 // - OpenCL spec defines several additional features, like, for example, 'v'
56 // modifier which allows to print OpenCL vectors: note that these features are
57 // not available on host device and therefore their usage should be either
58 // guarded using __SYCL_DEVICE_ONLY__ preprocessor macro or avoided in favor
59 // of more portable solutions if needed
60 //
61 template <typename... Args>
62 int printf(const CONSTANT_AS char *__format, Args... args) {
63 #ifdef __SYCL_DEVICE_ONLY__
64  return __spirv_ocl_printf(__format, args...);
65 #else
66  return ::printf(__format, args...);
67 #endif
68 }
