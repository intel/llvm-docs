Analyzes error code and arguments of urEnqueueKernelLaunch to emit user-friendly exception describing the problem.

This function is expected to be called only for non-success error codes, i.e. the first argument must not be equal to UR_RESULT_SUCCESS.

This function actually never returns and always throws an exception with error description.
