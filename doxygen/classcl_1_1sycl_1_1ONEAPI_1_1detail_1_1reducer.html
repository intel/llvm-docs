Detailed Description

template<typename T, class BinaryOperation, typename Subst = void>
class cl::sycl::ONEAPI::detail::reducer< T, BinaryOperation, Subst >

Class that is used to represent objects that are passed to user's lambda functions and representing users' reduction variable.

The generic version of the class represents those reductions of those types and operations for which the identity value is not known.
