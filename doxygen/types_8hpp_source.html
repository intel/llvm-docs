79 // All the captures of a lambda or functor of type FuncT passed to a kernel
80 // must be is_device_copyable, which extends to bases and fields of FuncT.
81 // Fields are captures of lambda/functors and bases are possible base classes
82 // of functors also allowed by SYCL.
83 // The SYCL-2020 implementation must check each of the fields & bases of the
84 // type FuncT, only one level deep, which is enough to see if they are all
85 // device copyable by using the result of is_device_copyable returned for them.
86 // At this moment though the check also allowes using types for which
87 // (is_trivially_copy_constructible && is_trivially_destructible) returns true
88 // and (is_device_copyable) returns false. That is the deprecated behavior and
89 // is currently/temporarily supported only to not break older SYCL programs.
90 template <typename FuncT>
91 struct CheckDeviceCopyable
92  : CheckFieldsAreDeviceCopyable<FuncT, __builtin_num_fields(FuncT)>,
93  CheckBasesAreDeviceCopyable<FuncT, __builtin_num_bases(FuncT)> {};
