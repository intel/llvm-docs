template <typename T, int N, typename AccessorTy, typename PropertyListT = oneapi::experimental::empty_properties_t> simd<T, N> mask_compress_store(AccessorTy acc, simd<T, N> vals, simd_mask<N> mask, PropertyListT props = {}); Mask compressed store to accessor memory (could be local or device accessor).

The function stores data to a memory location using following algorithm:

int Index = 0;
for (int i = 0; i < N; ++i) {
if (Mask[i])
acc[global_offset + Index++] = val[i];
}
